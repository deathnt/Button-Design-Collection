# Filling manganese button
**Author:** kisshka
**Created:** 03/11/2025

## Description
A transparent button that partially fills with pink shimmering liquid when hovered over and fully fills when clicked

## Features
- The effect of the rising liquid when you hover over the button
- Gradually filling the button while holding it

## Technologies
- HTML5
- CSS3
